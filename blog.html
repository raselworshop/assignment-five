What is the Document Object Model (DOM)?

DOM is a hierarchical representation of an html, consist of a root node is the document itself. it can change anthing on html page, like styles, tags etc.

How do you select an element from the DOM?

We can select an element using in a varities way:
a. getElementById().
b. getElementByTagName()
c. getElementByClassName().
d. querySelector(). and
e. querySelectorAll().

What is event delegation in the context of the DOM, and why is it useful?

We use event delegation to:
Manage event efficiently. without adding event listener to multiple child elements.
we can use event listner to parent element and take advantages of event bubbling to handle child elements
